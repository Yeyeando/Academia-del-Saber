# ¿Qué es y para qué sirve Academia del saber?.
Es una página donde puedes comprar cursos.
## Entornos usados
Laravel 12
XAMPP 
VS Code 
máquina virtual.
## Rutas
/: el welcom
/ecomarket: inicio del ecomarket
/hola: un saludo
/cursos: lista de cursos
## Vistas
welcome.blade.php: Muestra el mensaje de bienvenida a Laravel.
cursos.blade.php: Muestra la lista de cursos
   
## Comandos
php artisan route:list: veo todas las rutas del proyecto
